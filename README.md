# Пример конфигурации Ansible для управления инфраструктурой

## Запуск плейбука

Чтобы запустить плейбук, предварительно необходимо:

1. Установить Ansible на управляющий хост;
2. Создать пару SSH-ключей для подключения Ansible к управляемой инфраструктуре;
3. Убедиться, что имя и фактическое расположение приватного ключа совпадает с тем, что указано в `ansible.cfg`.

Запускается плейбук следующей командой:

```
ansible-playbook playbook.yaml 
```
